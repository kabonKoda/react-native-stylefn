# react-native-stylefn

A Tailwind-inspired styling library that injects design tokens into every React Native component's style props automatically

## Installation


```sh
npm install react-native-stylefn
```


## Usage


```js
import { multiply } from 'react-native-stylefn';

// ...

const result = await multiply(3, 7);
```
